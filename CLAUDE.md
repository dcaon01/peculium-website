# Progetto: Peculium

## Descrizione del progetto
Peculium è una startup fintech italiana che aiuta le persone a costruire e proteggere il proprio patrimonio sviluppando una reale consapevolezza finanziaria. Non un semplice tracker, ma un vero coach finanziario digitale: Peculium parte sempre dalla domanda "cosa vuoi ottenere?" e poi costruisce il percorso, lo monitora e lo adatta nel tempo. Un'app fintech italiana all-in-one per educazione finanziaria, gestione e decision support.

## Organizzazione del progetto
Per ora è un semplice sito "coming soon" da 1 pagina.

## Stile del codice
- Usa TypeScript strict mode
- Usa Prettier per la formattazione con la struttura che già vedi nei file esistenti:
  - per prime le importazioni con inject come prima cosa;
  - per seconde le dichiarazioni delle variabili da utilizzare nel componente;
  - costruttore o lifecycle functions;
  - funzioni varie.

## Note importanti
- Non modificare mai i file nella directory `libs/`, in cui ci sono i componenti spartan  
- Prima di modificare file di configurazione (es. tsconfig.json, package.json, .env, ecc.), segnalami quale file e perché, poi attendi il mio permesso esplicito.
- Per tutte le altre operazioni non chiedermi conferma se non esplicitato nel prompt.

## Una volta finito con un prompt
Una volta finito con un prompt manda una notifica a windows indicando che hai finito. Prediligi i dialog piuttosto che le notifiche di sistema.
